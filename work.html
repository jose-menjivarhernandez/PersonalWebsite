Stock Return Predictions through Multi-Factor Modelling,
and comparison with GRU ML results.

For SENG 550 - Large Scale Data Systems Engineering, my teammates and I
built a python-based application that had the ability to pull historical
pricing any random subset of stocks in the S&P 500 index. We then used this
data to run a linear regression ML algorithm. The features used were the financial
"factors", which according to financial theory, can explain stock returns.
The labels were price returns on the individual stocks. We then used a
gated recurring unit Neural Net only with previous historical prices
as features for future stock predictions and compared the results.

Technologies used:

  • Python
  • Databricks and Apache Spark
  • Pandas
  • SciKit Learn
  • PyTorch
